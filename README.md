# GTA 3 - Dunizzm DLC

Добро пожаловать в неофициальное дополнение (DLC) для Grand Theft Auto III. Этот мод преображает визуальную и техническую составляющую игры, адаптируя её под современные компьютеры и сохраняя оригинальную атмосферу.

## 🌟 Основные изменения

* **Новая модель Клода** — Теперь у главного героя новая одежда.
* **Улучшенная графика** — Обновленные текстуры окружения, дорог и транспорта, а также обновленные визуальные эффекты.
* **Современное меню** — Переработаны текстуры в главном меню, теперь оно выглядит лучше.
* **HD Радар** — Четкая карта и иконки на мини-карте в высоком разрешении.
* **Widescreen Fix** — Полноценная поддержка широкоформатных мониторов (исправлены пропорции экрана и интерфейса).
* **Исправление багов** — Устранены критические вылеты, проблемы с физикой при высоком FPS и старые графические артефакты.
* **Локализация** — Полная поддержка качественного русского языка (текст и шрифты).

## 🛠 Установка

1. Убедитесь, что у вас установлена чистая версия GTA 3.
2. Скачайте архив с модификацией ВО ВКЛАДКЕ RELEASE.
3. Скопируйте все файлы и папки из архива в корневую директорию игры (туда, где находится файл `gta3.exe`).
4. Подтвердите замену файлов, если система выдаст запрос.
5. Запустите игру.

## ⚙️ Системные требования

* **ОС:** Windows 7 / 8 / 10 / 11 (64-bit)
* **Процессор:** Intel Core 2 Duo / AMD FX или лучше
* **Оперативная память:** 3 GB RAM
* **Видеокарта:** С поддержкой DirectX 9.0c и 1 GB видеопамяти
* **Место на диске:** ~2 GB
